## Options

### blockPrefix

Type: `String`<br>
Required: `false`<br>
Default: `empty`

`prefix-block`

### elementPrefix

Type: `String`<br>
Required: `false`<br>
Default: `__`

`block__element`

### modifierPrefix

Type: `String`<br>
Required: `false`<br>
Default: `--`

`block__element--modifier`
